Cracked Phone

Spidered from the top corner, the crack pulls toward the power button like it fell face-first. The last brightness setting is still low, as if she didn’t want to be seen.

Recent calls show a number with no name, repeated, then deleted, then called again. The voicemail icon blinks, unheard.

A front camera smear suggests a hand that wasn’t steady… or one that didn’t want to be seen either.
